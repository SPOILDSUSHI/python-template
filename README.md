# How to use this

### 1. Fork the repo

### 2. Install user level dependencies

`pip install nox pre-commit`

### 3. Set up pre-commit

`pre-commit install`

### 4. Write great code with great tests!

### 5. Use Nox for testing

`nox`
`nox -rs tests`
